# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.1.3   | ✅ Yes     |
| < 2.1   | ❌ No      |

## Reporting a Vulnerability

**Please do not open public issues for security vulnerabilities.**

If you discover a security issue in VOID, contact us privately:

- Via the contact form at [void.toutakun04.qzz.io](https://void.toutakun04.qzz.io)

We aim to respond within **72 hours** and will keep you updated on the fix timeline.

## Scope

The following are in scope:

- APK signature or integrity issues
- E2EE implementation flaws
- Auth bypass or token leakage
- Server-side data exposure
- Any issue that could compromise user privacy

## Out of Scope

- Bugs in outdated/unsupported versions
- Issues requiring physical device access
- UI/UX bugs (report these as regular issues)
